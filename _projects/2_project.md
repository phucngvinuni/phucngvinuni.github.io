An AI-powered smart shopping cart system enabling hands-free shopping experiences, developed as a Top 20 Finalist in the **IoT Challenge 2025 (FPT Software)**.

**Key Features:**
- Real-time shopper positioning via Machine Learning on Silicon Labs hardware
- Autonomous navigation with A* pathfinding algorithm
- Quantized LLMs on Raspberry Pi for conversational assistant
- Full-stack Frontend/Backend for real-time tracking and route optimization

**Technologies:** Python, Edge AI, Raspberry Pi, Silicon Labs, LLM quantization
